## 1. The "What If" Chain

The most powerful brainstorming tool. Start with a seed and branch:

**Process:**

1. Start with ONE "What if" question
2. For each answer, generate 3 more "What if" follow-ups
3. Go 3-4 levels deep
4. Circle the ideas that create the strongest emotional reaction

**Example Chain:**

```
What if a hit man had to protect his target?
├── What if the target was a child who witnessed a crime?
│   ├── What if the child doesn't realize what they saw?
│   ├── What if the hit man was hired by the child's parent?
│   └── What if the child reminds the hit man of his own kid?
├── What if the target hired the hit man themselves?
│   ├── What if it's an assisted suicide that goes wrong?
│   └── What if they're trying to fake their own death?
└── What if the hit man discovers the contract on the target is a setup?
    ├── What if the real target is the hit man himself?
    └── What if completing the job triggers a larger catastrophe?
```

**AI Prompt Template:**

```
Generate 10 "What if" premise starters for a [GENRE] film 
targeting [AUDIENCE]. Each should be:
- High concept (explainable in one sentence)
- Emotionally resonant (taps into universal fear/desire)
- Commercially viable (imagine the trailer)

Then for the 3 strongest, branch 3 levels deep with follow-up 
"What if" questions that complicate the story.
```

## 2. Logline Workshop

Systematically develop and refine loglines through iteration.

**The Logline Components:**

1. **Protagonist** — Specific, vivid descriptor (not just "a man")
2. **Inciting Incident** — What disrupts their world
3. **Goal** — Active, concrete objective
4. **Obstacle** — Primary antagonistic force
5. **Stakes** — What happens if they fail
6. **Irony/Hook** — The element that makes it fresh

**Iteration Protocol:**

```
Round 1: Write 5 loglines for the concept (breadth)
Round 2: Pick the 2 strongest, write 3 variations of each (depth)
Round 3: Stress-test the winner with these questions:
  - Would you click on this Netflix thumbnail?
  - Can you see the poster?
  - What actor plays the lead?
  - What's the comp title? ("It's [X] meets [Y]")
Round 4: Final polish — every word earns its place
```

**AI Prompt Template:**

```
I have a concept: [CONCEPT]

Write 5 loglines, each emphasizing a different aspect:
1. Focus on the protagonist's internal conflict
2. Focus on the external stakes
3. Focus on the ironic hook
4. Focus on the emotional promise
5. Focus on the ticking clock/urgency

For each, format as: When [INCITING], a [PROTAGONIST] must 
[GOAL] before [DEADLINE/STAKES], but [OBSTACLE].
```

## 3. The 100 Ideas Exercise

Quantity breeds quality. Generate volume, then filter ruthlessly.

**Process:**

1. Set a topic (e.g., "thriller concepts," "ways this heist could go wrong")
2. Generate 100 ideas — no filtering, no judgment
3. Rate each 1-5 on ORIGINALITY and EMOTION
4. Keep only the 5s (usually 5-10 ideas)
5. Combine surviving ideas for unexpected mashups

**AI Prompt Template:**

```
Generate 25 ideas for [TOPIC]. Rules:
- No clichés (if you've seen it in 3+ movies, skip it)
- Each must create a visceral emotional reaction
- Mix tones: some dark, some funny, some heartbreaking
- Vary scale: some intimate, some epic
- At least 5 should be ideas you've NEVER seen in a film

Then flag the top 5 with the strongest commercial potential 
and explain WHY each would attract an audience.
```

## 4. Character Concept Mining

Build characters from the inside out before fitting them into plot.

**Character Seed Questions:**

1. What's the WORST thing that could happen to this specific person?
2. What secret are they keeping from everyone?
3. What contradiction defines them? (e.g., a pacifist soldier, a honest con artist)
4. What do they want? What do they ACTUALLY need?
5. What's the lie they believe about themselves?

**AI Prompt Template:**

```
Create 5 protagonist concepts for a [GENRE] film. For each:

CHARACTER: [Vivid one-line description with contradiction]
WOUND: [The past event that shaped them]
LIE: [The false belief they operate under]
WANT: [What they're actively pursuing]
NEED: [What they actually require for fulfillment]
FLAW: [The behavior pattern that sabotages them]
GIFT: [The unique strength only they possess]

Make each character someone an A-list actor would kill to play. 
Focus on compelling contradictions.
```

## 5. World-Building Brainstorms

Create vivid, specific worlds that enhance the story.

**World-Building Prompt Template:**

```
Build the world for a [GENRE] set in [SETTING]. Include:

RULES: What's different about this world? What are the constraints?
TEXTURE: 5 specific sensory details (sights, sounds, smells, textures, tastes)
CULTURE: Social norms, hierarchies, rituals
CONFLICT: Built-in tensions in this world
SECRETS: What's hidden beneath the surface?
VISUAL SIGNATURE: What makes this world look like no other film?

Focus on details that CREATE STORY — every world fact should 
generate potential conflict or character moments.
```

## 6. Premise Stress-Testing

Before committing to a premise, subject it to rigorous testing.

**The 10 Stress Tests:**

| Test | Question | Red Flag |
|------|----------|----------|
| Audience | Who pays to see this? | "Everyone" (too vague) |
| Freshness | What's truly new here? | "It's like X but..." (derivative) |
| Emotion | What feeling does this create? | None specifically (intellectual only) |
| Protagonist | Active or passive? | Things happen TO them |
| Stakes | What if they fail? | Nothing irreversible |
| Castability | Who plays the lead? Why? | No one specific comes to mind |
| Budget | What does this cost? | Impossible for the story's market |
| Title | Can you name it? | Nothing sticks |
| Trailer | Can you cut a trailer? | No visual moments |
| Dinner Table | Would you tell a friend about this? | It's interesting... but forgettable |

**AI Prompt Template:**

```
Stress-test this premise: [PREMISE]

For each of these 10 dimensions, rate it 1-5 and explain:
1. Audience clarity
2. Freshness/originality
3. Emotional power
4. Protagonist agency
5. Stakes clarity
6. Castability
7. Budget feasibility
8. Title potential
9. Trailer potential
10. Word-of-mouth potential

Then prescribe the 3 most important improvements to make 
this premise commercially stronger.
```

## 7. Genre Mash-Up Generator

Some of the best films combine unexpected genres.

**Process:**

1. Pick 2-3 genres that don't usually go together
2. Ask: "What if [GENRE A] had the structure of [GENRE B]?"
3. Find the emotional throughline that unifies them

**Examples of Successful Mash-ups:**

- *Get Out* = Horror + Social Satire
- *Parasite* = Thriller + Dark Comedy + Class Drama
- *Everything Everywhere All at Once* = Sci-Fi + Family Drama + Martial Arts Comedy
- *The Lobster* = Romance + Dystopian Satire

**AI Prompt Template:**

```
Mash up [GENRE A] with [GENRE B]. Generate 5 premises where:
- The genre combination feels surprising but inevitable
- The emotional core is clear and universal
- The concept is high-concept (one-sentence pitch)
- A specific audience would champion this film
```

## 8. The Conflict Matrix

Generate conflict systematically by crossing characters with obstacles.

**Matrix Structure:**

```
             | Internal     | Interpersonal | Societal      | Environmental
Protagonist  | Self-doubt   | Betrayal      | Oppression    | Survival
Antagonist   | Regret       | Rivalry        | Corruption    | Scarcity
Ally         | Secret       | Loyalty test   | Complicity    | Sacrifice
```

Fill each cell with a specific conflict for YOUR story. The richest screenplays have conflict in multiple cells simultaneously.

**AI Prompt Template:**

```
For this story concept: [CONCEPT]

Create a conflict matrix with these characters: [CHARACTERS]

For each character, generate specific conflicts across all 4 
dimensions (Internal, Interpersonal, Societal, Environmental).

Then identify the 3 most cinematically compelling conflict 
intersections — moments where multiple dimensions collide.
```

    **2. Create src/components/bulk/AddToFavoritesModal.tsx (D-05, BULK-09):**

    ```typescript
    /**
     * AddToFavoritesModal
     * Adds all selected screenplays to a chosen favorites list.
     * Shows Quick Favorites as default option, plus any user-created lists.
     */
    ```

    Props interface:
    ```typescript
    interface AddToFavoritesModalProps {
      isOpen: boolean;
      onClose: () => void;
    }
    ```

    Component implementation:
    - Use `useState<string>('quick')` for `selectedList` state -- 'quick' for Quick Favorites, list ID for named lists
    - Call `useFavoritesStore((s) => ({ lists: s.lists }))` to get user lists
    - Call `useSelectionStore((s) => s.selectedIds)` to get selected IDs
    - If `!isOpen` return null

    Apply handler:
    ```typescript
    const handleApply = () => {
      const ids = Array.from(selectedIds);
      if (selectedList === 'quick') {
        // For Quick Favorites, only add those not already quick-favorited
        for (const id of ids) {
          if (!useFavoritesStore.getState().isQuickFavorite(id)) {
            useFavoritesStore.getState().toggleQuickFavorite(id);
          }
        }
        useToastStore.getState().addToast(
          `Added ${ids.length} to Quick Favorites`,
          'success'
        );
      } else {
        for (const id of ids) {
          useFavoritesStore.getState().addToList(selectedList, id);
        }
        const listName = useFavoritesStore.getState().lists.find(l => l.id === selectedList)?.name || 'list';
        useToastStore.getState().addToast(
          `Added ${ids.length} to ${listName}`,
          'success'
        );
      }
      onClose();
    };
    ```

    JSX layout per UI-SPEC:
    - Same modal overlay pattern as SetCategoryModal (z-50, backdrop, glass panel, animate-scale-in)
    - Header: "Add to Favorites" with `text-lg font-heading font-semibold text-gold-200`
    - Body: `px-6 py-4 space-y-3`
      - Quick Favorites row (always first):
        - `flex items-center gap-3 p-3 rounded-lg cursor-pointer transition-colors`
        - When selected: `bg-gold-500/10 border border-gold-500/30`
        - When not selected: `bg-black-800/50 border border-black-700 hover:border-black-600`
        - Star SVG icon: `w-5 h-5 text-gold-400` (use star path: `M12 2l3.09 6.26L22 9.27l-5 4.87 1.18 6.88L12 17.77l-6.18 3.25L7 14.14 2 9.27l6.91-1.01L12 2z`)
        - Label: `text-sm text-gold-200`, "Quick Favorites"
        - Hidden radio input: `<input type="radio" name="favorites-list" value="quick" checked={selectedList === 'quick'} onChange={() => setSelectedList('quick')} className="sr-only" />`
        - The entire row is clickable via `onClick={() => setSelectedList('quick')}`
      - Named lists (map from `lists`):
        - Same row pattern, label uses `text-sm text-black-200`, list name as text
        - Radio: `value={list.id}`, `checked={selectedList === list.id}`, `onChange={() => setSelectedList(list.id)}`
        - Visual indicator for selected: right side shows a filled circle or checkmark. Simplest: use a small `w-4 h-4 rounded-full` div that is `bg-gold-500` when selected, `bg-black-600` when not.
    - Footer: `px-6 py-4 flex justify-end gap-3`
      - Cancel button: `btn btn-ghost text-sm`, text "Keep Favorites", onClick={onClose}
      - Apply button: `btn btn-primary text-sm`, text "Add to Favorites", onClick={handleApply}

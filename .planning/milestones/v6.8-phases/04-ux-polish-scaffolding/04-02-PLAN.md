  <action>
    **Part A: safeJsonParse utility**

    Add to `src/lib/utils.ts` (file already exists — append, do not overwrite):

    ```typescript
    export function safeJsonParse<T>(raw: string | null | undefined, fallback: T): T {
      if (raw == null) return fallback;
      try {
        return JSON.parse(raw) as T;
      } catch {
        return fallback;
      }
    }
    ```

    **Part B: safeJsonParse tests**

    Create `src/lib/utils.test.ts` (or append if it exists). Tests:

    - Test: undefined input returns the fallback value
    - Test: empty string returns the fallback value
    - Test: valid JSON array returns parsed array
    - Test: fallback type is preserved (number fallback returns number)

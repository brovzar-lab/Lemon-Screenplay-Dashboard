# Plan 01-01: Fix Coverage PDF Cover Page Score/Verdict Spacing

## Objective

Fix the known visual defect on the coverage PDF cover page where the weighted score number and recommendation badge (RECOMMEND/CONSIDER/PASS) are visually merged. Add proper spacing between them.

## Context

The cover page `scoreLeft` column renders three stacked elements:
1. `scoreNum` — large 32pt score number (e.g., "7.8")
2. `scoreOf` — small "out of 10" text
3. `recBadge` — colored recommendation badge

Current spacing: `scoreNum` marginBottom 2pt → `scoreOf` marginBottom 6pt → `recBadge` with no top margin. The 6pt gap between "out of 10" and the badge makes them look merged.

In `src/components/export/CoverageDocument.tsx`, modify the styles in the score hero section:

1. Change `scoreOf` marginBottom from `6` to `10`:
   ```
   scoreOf: { fontSize: 7, color: C.grey500, marginBottom: 10 },
   ```

2. Add `marginTop: 4` to `recBadge`:
   ```
   recBadge: { marginTop: 4, paddingVertical: 3, paddingHorizontal: 12, borderRadius: 2 },
   ```

This creates 14pt total separation (10 + 4) between the "out of 10" text and the recommendation badge, making them visually distinct while keeping the left column compact.
